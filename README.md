# 💸 Bill Splitting App

A full-stack web application to split bills equally among a group of people.

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Java, Spring Boot
- **Database:** H2 (In-memory SQL database)

## ✨ Features
- Add and remove people from a group
- Add expenses with description and amount
- Track who paid for each expense
- Auto-calculate how much each person owes or receives
- Clean and responsive UI

## 🚀 How to Run
1. Clone the repository
2. Open backend in IntelliJ and run BillsplitApplication.java
3. Open frontend/index.html in browser
4. App runs on http://localhost:8080
